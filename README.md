# SOLO-GIGANEKIT-PROJECT-2026

## Описание проекта
Данный репозиторий создан в рамках выполнения **Практической работы №1 - #7** по теме «Создание проекта и репозитория в GitHub».

Цель проекта — освоение базового рабочего процесса (workflow) в Git:
* Клонирование репозитория.
* Работа с ветками (`feature/`).
* Фиксация изменений через коммиты.
* Создание Pull Request для слияния кода в основную ветку.

## Используемые технологии
* **Git** — система контроля версий.
* **GitHub** — облачная платформа для хостинга.
* **GitHub Projects** — планирование задач на Kanban-доске.
